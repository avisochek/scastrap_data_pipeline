"mcl" folder contains the Markov Clustering Algorithm[1]. The algorithm is called by typing the following in the command line:

            mcl inputfilename --abc -o outputfilename

autocluster.sh automates mcl for each .tsv in a directory. autocluster.sh iterates through the directory for any files ending in .tsv and 
runs mcl on each tsv, overwriting the original tsv with the clustered file.

          example@example ~/foo $ touch one.tsv two.tsv three.tsv
          example@example ~/foo $ bash autocluster.sh
 

autocluster.sh iterates through the directory and does the following:

          mcl one.tsv --abc -o one.tsv
          mcl two.tsv --abc -o two.tsv
          mcl three.tsv --abc -o three.tsv

It overwrites the original files (one.tsv, two.tsv, three.tsv) with the clustered files.


----------------------------------------------------------------------------------------------------------------------------------          


pcluster.py (stands for Process Cluster) takes each clustered file and transposes the individual clusters. It takes the 
clustered file generated by mcl in format:

item1,item2,item3,item4
item5,item6,item7

And changes it to:

cluster1
item1
item2
item3
item4


cluster2
item5
item6
item7

This separates the clusters in an easily visualizable way and labels each cluster for easy computer input.
